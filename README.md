# 도서관리 프로그램 + 열람실 및 카드활용

설치: 
1. localhost:3306포트로 mysql 환경 설치
2.  dataset 폴더 내 sql 임포트 시켜서 설치
3. 아두이노 컴퓨터 연결 후, 포트 확인(ex: com6)
4. Main.cs 파일에서 port 및 mysql 로그인 정보 변경.
5. 실행
